## 3. Цветовая палитра

Направление: **премиальный тёмный + акцентный красный** (автомобильная классика: карбон, лак, стоп-сигнал).

| Роль | Цвет | HEX | Применение |
|------|------|-----|------------|
| Фон основной | Карбон | `#0B0F14` | фон сайта (dark-first) |
| Фон поверхностей | Графит | `#141A22` | карточки, секции |
| Фон приподнятый | Сталь | `#1E2732` | ховеры, бордеры-подложки |
| Текст основной | Белый лак | `#F4F6F8` | заголовки, основной текст |
| Текст вторичный | Серебро | `#98A4B3` | подписи, описания |
| **Акцент / CTA** | Сигнальный красный | `#E11D2E` | кнопки, ссылки, логотип |
| Акцент ховер | Красный тёмный | `#B91626` | hover-состояния |
| Бордер | Тёмная сталь | `#26303C` | разделители, обводки |
| Успех | Зелёный | `#2FBF71` | статусы «готово», чеки |

### CSS-токены

```css
:root {
  --bg: #0B0F14;
  --surface: #141A22;
  --surface-2: #1E2732;
  --text: #F4F6F8;
  --text-muted: #98A4B3;
  --accent: #E11D2E;
  --accent-hover: #B91626;
  --border: #26303C;
  --success: #2FBF71;
}
```

### Правила
- Красный — только для действий и акцентов, не более 10% площади экрана
- Контраст текста ≥ 4.5:1 (проверено: #F4F6F8 на #0B0F14 = 17:1, #98A4B3 на #0B0F14 = 7.2:1)
- Никаких чистых `#000000` и `#FFFFFF` — только тональные варианты

## 5. Графика и эффекты

**Стиль:** тёмный премиум с элементами liquid glass — полупрозрачные поверхности, мягкий blur, тонкие градиентные свечения.

- Карточки: `background: var(--surface)`, бордер 1px `var(--border)`, радиус **16px**
- Кнопки: радиус **12px**, высота ≥ 48px
- Свечение акцента: `box-shadow: 0 0 40px rgba(225,29,46,.25)` — дозированно, у CTA и hero
- Декоративный градиент: красное радиальное свечение на тёмном фоне (имитация света фар)
- Анимации: 200–400ms, `ease-out` на появление; плавный reveal секций при скролле
- Уважать `prefers-reduced-motion`

**Иконки:** только SVG (Lucide/Heroicons, stroke 1.5–2px). Эмодзи как иконки — запрещены.

**Фото:** тёмные, контрастные, с бликами на лаке. Автомобили — крупные планы деталей: фара, капот, кожа салона. Никаких стоковых улыбающихся механиков.
